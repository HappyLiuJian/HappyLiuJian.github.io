JavaScript简明学习(一) 历史

JavaScript的出现

最开始的网页表单验证需要经过后台服务器，时间久，效率低，前端脚本语言被发明解决这个问题

JavaScript的实现

ECMAScript+DOM+BOM

JavaScript简明学习(二) 在HTML中使用

Script标签

标签属性

  1. async可选，表示立即加载
  2. charset可选，表示设定字符集，一般忽略
  3. defer可选，脚本可延迟到文档完全解析之后再执行
  4. src可选，外部js文件位置
  5. type可选，制定脚本类型 text/javascript

使用方式为嵌入或者包含外部文件

imp显示</script>需要转义，如：

<script type = "text/javascript">
    function sayScript() {
        alert("<\/script>");
    }
</script>

否则会遇到alert中的</script>认为是结束标记

标签的位置

<script>标签的位置放在最后的</body>之前，保证页面先呈现内容再加载脚本

延迟脚本 外部脚本添加defer属性形成 脚本本身在遇到</html>之后执行，若有多个延迟脚本，按照书写顺序执行脚本
异步脚本 外部脚本添加async属性形成 脚本本身在遇到</html>之后执行，若有多个延迟脚本，不一定按照书写顺序执行脚本 一定在window.onload之前执行，但可能在DOMContentLoaded之前或之后执行

<noscript>标签用以在不支持脚本或者脚本禁用的浏览器中平稳退化，显示替代内容，如：

<!DOCTYPE html>
<html>
<head>
    <title>Example</title>
    <script type="text/javascript" src="jquery.js"></script>
</head>
<body>
    <noscipts>
        <p>本页面需要浏览器支持</p>
    </noscipts>
</body>
</html>
