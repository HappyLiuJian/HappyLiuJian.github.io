JavaScript基本概念

语法

  1. 严格区分大小写
  2. 标识符首字符为字母、下划线或一个美元符，其他字符可以包括数字
  3. 注释为c风格

imp严格模式 使用”use strict”进入严格模式，”use strict”可以在顶部或函数中使用

语句

句末分号可有可无，但推荐有

关键字和保留字

关键字和保留字不能作为标识符
关键字作标识符，报”Identifier Expected”错误，保留字是否有错取决于引擎
在严格模式下，保留字一定报错

变量

变量为松散类型，关键字为var，可以保存任何类型数据

var message = "hi";
message = 100;

完成更改操作(不建议更改变量值)

function test() {
    var message = "hi";
}
test();
alert(message);

报错，因为函数内定义的var变量为局部变量，函数退出后会销毁

function test() {
    message = "hi"
}
test();
alert(message);

正确，不使用var定义的变量为全局变量(不推荐，容易混乱并且不易维护)

数据类型

5种基本数据类型(Undefined,Null,Boolean,Number,String)和一种负载类型(Object)
typeof操作可以返回数据类型

typeof message;
typeof(message);
typeof 95;

typeof是一个操作符，不是函数，后面可以不用跟括号(但加上也不错)

Undefined

当一个变量未初始化值时，其类型即为undefined
一个声明但没有初始化的变量和一个未声明的变量是有区别的

var message;
//var age;

alert(message);
alert(age);

前者会警告变量值为undefined,后者会报错，提示没有变量

Null

null是一个空对象指针,使用typeof操作返回一个object值
推荐显示设置某个变量为null，用来保存对象
imp

alert(null == undefined)

这句话返回值为true

Boolean

true或者false(True和False并不是它的值)
任何类型的值可以通过Boolean函数转化为bool值

var message = "hi";
var BOOlmessage = Boolean(message);

转化的关系表如下：
类型 true false
Boolean true false
String 任何非空 “”
Number 任何非零 0和NaN
Object 任何对象 null
Undefined n/a undefined

Number

十进制，八进制(前面加0，如果超出范围，视为10进制)，十六进制(前面加0x)

var intNum = 10;    //十进制

var octalNum = 070;    //八进制
var octalNum = 079; //无效的八进制，视为十进制，79

var hexNum = 0xA;    //十进制的10

浮点数值 保存空间为整数值的两倍

var floatNum = 1.1;
var floatNum = .1;    //有效，不推荐

var floatNum = 1.0;    //会被自动解析为1以节省空间

var floatNum = 3.12e7;    //等于31200000
var floatNum = 3.12e-17;    //等于0.00000000312

数值范围5e-324(Number.MIN_VALUE)到1.7976931348623157e+308(Number.MAX_VALUE)
超出转换为Infinity或者-Infinity
函数isFinity()用来判断是否是有限数

NaN 非数值，表示本来需要返回数值的操作数未返回数值(不会再抛出错误)，如：任何数值除以0，返回一个NaN，代码可以继续运行
注：任何涉及NaN的操作，都返回NaN，NaN与任何值不等，包括自身

alert(NaN == NaN);    //false

isNaN()函数判断是否不是数值或者能转换为不是数值，true表示不是数值.如：

alert(isNaN(NaN));    //true
alert(isNaN(10));    //false
alert(isNaN("10"));    //false,"10"可以转化为10
alert(isNaN("blue"));    //true
alert(isNaN(true));    //false,可以转化为1

三个数值转换函数 Number(),parseInt(),parseFloat()
Number()用于所用数据类型

JavaScript简明学习(一) 历史

JavaScript的出现

最开始的网页表单验证需要经过后台服务器，时间久，效率低，前端脚本语言被发明解决这个问题

JavaScript的实现

ECMAScript+DOM+BOM

JavaScript简明学习(二) 在HTML中使用

Script标签

标签属性

  1. async可选，表示立即加载
  2. charset可选，表示设定字符集，一般忽略
  3. defer可选，脚本可延迟到文档完全解析之后再执行
  4. src可选，外部js文件位置
  5. type可选，制定脚本类型 text/javascript

使用方式为嵌入或者包含外部文件

imp显示</script>需要转义，如：

<script type = "text/javascript">
    function sayScript() {
        alert("<\/script>");
    }
</script>

否则会遇到alert中的</script>认为是结束标记

标签的位置

<script>标签的位置放在最后的</body>之前，保证页面先呈现内容再加载脚本

延迟脚本 外部脚本添加defer属性形成 脚本本身在遇到</html>之后执行，若有多个延迟脚本，按照书写顺序执行脚本
异步脚本 外部脚本添加async属性形成 脚本本身在遇到</html>之后执行，若有多个延迟脚本，不一定按照书写顺序执行脚本 一定在window.onload之前执行，但可能在DOMContentLoaded之前或之后执行

<noscript>标签用以在不支持脚本或者脚本禁用的浏览器中平稳退化，显示替代内容，如：

<!DOCTYPE html>
<html>
<head>
    <title>Example</title>
    <script type="text/javascript" src="jquery.js"></script>
</head>
<body>
    <noscipts>
        <p>本页面需要浏览器支持</p>
    </noscipts>
</body>
</html>

—end—
