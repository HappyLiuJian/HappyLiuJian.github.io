JavaScript基本概念（续）

操作符

ECMAScript的操作符作用于多种类型，包括字符串、数字、布尔，甚至对象

一元操作符

++、—: 和c相同
+、-：+表示正、-表示负

位操作符

按位非：~ 结果为求负数再减一

var a = 25;
var b = ~a;    //b的值-26

按位与：&
按位或：|
按位异或：^
左移：<< 左移不会影响符号

var a = 2;
var b = a << 5;        //b为64，参数为移动位数，1表示2倍

有符号右移：>> 保留符号 右移后以符号位填充
无符号右移：>>> 不保留符号 右移后以0填充

imp使用移位操作进行乘除计算效率比正常计算高

布尔操作符

逻辑非：！
逻辑与：&&
逻辑或：||

乘性操作符

  • / %

加性操作符

  • -

关系操作符

< > <= >=

等性操作符

== !=
=== !== 全等和不全等

条件操作符

a = b ? c : d

赋值操作符

= *= /= += -= <<= >>=

语句

if、do-while、for、break和continue、switch都与c相同

for-in语句：精准的迭代语句，用来枚举对象属性，如下：

for(var propName in window) {
    document.write(propName);
}

对于window中的每一个属性，赋值给变量propName，直到window中的所有属性被枚举

lable语句：用来在代码中添加标签，如下：

lable: statement;    //语法
start: var a = 2;

with语句：将代码的作用域设置到特定对象中，简化多次编写同一对象的工作，如下：

with(expression) statement;        //语法

var a = location.search.toString();
var b = location.name;
var c = location.age;

with(location) {
    var a = search.toString();
    var b = name;
    var c = age;
}

函数

基本与c相同，定义方式如下：

function functionName(arg0,arg1...) {
    statements;
}

imp函数没有重载
