JavaScript基本概念

语法

  1. 严格区分大小写
  2. 标识符首字符为字母、下划线或一个美元符，其他字符可以包括数字
  3. 注释为c风格

imp严格模式 使用”use strict”进入严格模式，”use strict”可以在顶部或函数中使用

语句

句末分号可有可无，但推荐有

关键字和保留字

关键字和保留字不能作为标识符
关键字作标识符，报”Identifier Expected”错误，保留字是否有错取决于引擎
在严格模式下，保留字一定报错

变量

变量为松散类型，关键字为var，可以保存任何类型数据

var message = "hi";
message = 100;

完成更改操作(不建议更改变量值)

function test() {
    var message = "hi";
}
test();
alert(message);

报错，因为函数内定义的var变量为局部变量，函数退出后会销毁

function test() {
    message = "hi"
}
test();
alert(message);

正确，不使用var定义的变量为全局变量(不推荐，容易混乱并且不易维护)

数据类型

5种基本数据类型(Undefined,Null,Boolean,Number,String)和一种负载类型(Object)
typeof操作可以返回数据类型

typeof message;
typeof(message);
typeof 95;

typeof是一个操作符，不是函数，后面可以不用跟括号(但加上也不错)

Undefined

当一个变量未初始化值时，其类型即为undefined
一个声明但没有初始化的变量和一个未声明的变量是有区别的

var message;
//var age;

alert(message);
alert(age);

前者会警告变量值为undefined,后者会报错，提示没有变量

Null

null是一个空对象指针,使用typeof操作返回一个object值
推荐显示设置某个变量为null，用来保存对象
imp

alert(null == undefined)

这句话返回值为true

Boolean

true或者false(True和False并不是它的值)
任何类型的值可以通过Boolean函数转化为bool值

var message = "hi";
var BOOlmessage = Boolean(message);

转化的关系表如下：
类型 true false
Boolean true false
String 任何非空 “”
Number 任何非零 0和NaN
Object 任何对象 null
Undefined n/a undefined

Number

十进制，八进制(前面加0，如果超出范围，视为10进制)，十六进制(前面加0x)

var intNum = 10;    //十进制

var octalNum = 070;    //八进制
var octalNum = 079; //无效的八进制，视为十进制，79

var hexNum = 0xA;    //十进制的10

浮点数值 保存空间为整数值的两倍

var floatNum = 1.1;
var floatNum = .1;    //有效，不推荐

var floatNum = 1.0;    //会被自动解析为1以节省空间

var floatNum = 3.12e7;    //等于31200000
var floatNum = 3.12e-17;    //等于0.00000000312

数值范围5e-324(Number.MIN_VALUE)到1.7976931348623157e+308(Number.MAX_VALUE)
超出转换为Infinity或者-Infinity
函数isFinity()用来判断是否是有限数

NaN 非数值，表示本来需要返回数值的操作数未返回数值(不会再抛出错误)，如：任何数值除以0，返回一个NaN，代码可以继续运行
注：任何涉及NaN的操作，都返回NaN，NaN与任何值不等，包括自身

alert(NaN == NaN);    //false

isNaN()函数判断是否不是数值或者能转换为不是数值，true表示不是数值.如：

alert(isNaN(NaN));    //true
alert(isNaN(10));    //false
alert(isNaN("10"));    //false,"10"可以转化为10
alert(isNaN("blue"));    //true
alert(isNaN(true));    //false,可以转化为1

三个数值转换函数 Number(),parseInt(),parseFloat()
Number()用于所用数据类型

  1. Boolean，true和false分别转换为1和0
  2. 数字，传入返回
  3. null，返回0
  4. undefined，返回NaN
  5. 字符串时
  • 只有数字，变为十进制数字“011”也会变为11
  • 有效浮点数变为相应浮点数，如“1.1”变为1.1
  • 有效十六进制变为相应十六进制，如“0xf”
  • 字符串为空，转为0
  • 其他，转为null
  1. 当为对象时，调用其valueof()方法，然后按前面的规则转换。结果为NaN时，再调用对象的toString()方法，按前面的规则转换

parseInt()只对字符串起作用，会忽略字符串前的空格，找到第一个非空格字符，若不是数字，返回NaN，若是，继续查看下一个，直到找到非数字(转换能识别进制，并且只有整数，不支持小数点.识别)

parseFloat()识别小数，第一次扫描的.识别，第二次则不识别

String

特点：
单引号、双引号都可以，没有区别
字符串不可变，一旦创建不能改变值。要改变某个变量保存的字符串，必须销毁原来的字符串，如：

var lang = "java";
lang = lang + "script";

第二行代码的作用过程为创建一个10个字符长的变量，向其中填充”java”和”script”字符串，最后销毁原来的字符串”java”和”script”

字符串转换：
toString()方法：返回相应值得字符串表示；在转换可以有一个参数，表示基数(进制)，如：

var a = 8;
a.toString() = "8";
a.toString(2) = "1000"
a.toString(10) = "8"

默认无参数和参数为10效果相同
String()方法： 参数可以用toString()时与其一样，另：null转为”null”,undefined转为”undefined”

Object类型

数据和方法的集合

var 0 = new Object();

使用new+对象名称声明

在ECMAScript中，Object对象为所有对象的基础
