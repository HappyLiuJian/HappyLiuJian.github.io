EMMET的使用

在sublime中的安装

首先你需要为sublime text安装Package Control组件:

  • 按Ctrl+`调出sublime text的console

  • 粘贴以下代码到底部命令行并回车

    import urllib2,os;pf=’Package Control.sublime-package’;ipp=sublime.installedpackagespath();os.makedirs(ipp) if not os.path.exists(ipp) else None;open(os.path.join(ipp,pf),’wb’).write(urllib2.urlopen(‘http://sublime.wbond.net/'+pf.replace(‘ ‘,’%20’)).read())

  • 重启Sublime Text

  • 在Perferences->package settings中看到package control，则表示安装成功

使用Package Control安装Emmet插件:

  • 按Ctrl+Shift+P命令板
  • 输入pci然后选择install Package，然后输入emmet找到 Emmet Css Snippets，点击就可以自动完成安装

使用方法

直接在编辑器中输入HTML或CSS的代码的缩写，然后按tab键就可以拓展为完整的代码片段

语法：

后代 >
如: nav>ul>li

<div>
    <ul>
        <li></li>
    </ul>
</div>

兄弟 +
如: div+b+bq

<div></div>
<b></b>
<q></q>

上级 ^
如: div+div>p>q+em^bq

<div></div>
<div>
    <p><q></q><em></em></p>
    <blockquote></blockquote>
</div>

如: div+div>p>q+em^^bq

<div></div>
<div>
    <p><q></q><em></em></p>
</div>
<blockquote></blockquote>

分组 ()
如: div>(header>ul>li*2>a)+footer>p

<div>
    <header>
        <ul>
            <li><a href=""></a></li>
            <li><a href=""></a></li>
        </ul>
    </header>
    <footer>
        <p></p>
    </footer>
</div>

如: (div>dl>(dt+dd)*3)+footer>p

<div>
    <dl>
        <dt></dt>
        <dd></dd>
        <dt></dt>
        <dd></dd>
        <dt></dt>
        <dd></dd>
    </dl>
</div>
<footer>
    <p></p>
</footer>

乘法
如: ul>li5

<ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>

自增符号 $
如: ul>li.item$*5

<ul>
    <li class="item1"></li>
    <li class="item2"></li>
    <li class="item3"></li>
    <li class="item4"></li>
    <li class="item5"></li>
</ul>

如: h$[title=item$]{Header $}*3

<h1 title="item1">Header 1</h1>
<h2 title="item2">Header 2</h2>
<h3 title="item3">Header 3</h3>

如: ul>li.item$$$*5

<ul>
    <li class="item001"></li>
    <li class="item002"></li>
    <li class="item003"></li>
    <li class="item004"></li>
    <li class="item005"></li>
</ul>

如: ul>li.item$@-*5

<ul>
    <li class="item5"></li>
    <li class="item4"></li>
    <li class="item3"></li>
    <li class="item2"></li>
    <li class="item1"></li>
</ul>

ID和类属性
ID：#header

<div id="header"></div>

类: .title

<div class="title"></div>

自定义属性
p[title=”Hello world”]

<p title="Hello world"></p>

文本：{}
a{Click me}

<a href="">Click me</a>

所有未知的缩写都会转换成标签，例如，foo →
