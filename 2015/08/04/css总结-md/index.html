css总结.md

发表于 2015-08-04   |  
  1. 定宽元素水平居中 {width: xxpx; margin: xxpx auto;}
  2. 不定宽元素居中 {position:absolute top:xxpx right:xxpx ….} 其父辈是position: relative的
  3. 文字垂直居中 {height:xpx; line-height:xpx} 即高度与行间高度一致，这个方法对行内元素和会级元素都有用
    {padding:xxpx;} 外框元素设置padding值
或者采用position方法，进行精确定位
注意overflow方法的使用不然可能会导致内部元素溢出
  4. 默认样式一定要预先清除···一定要 *{margin:0; padding:0;}
  5. overflow:hidden特别有用···特别有用
  6. 先考虑好元素的语义结构很重要
  7. text-indent（文本缩进用，text-indent替换padding-left，是为了防止菜单长度增加。例如：padding-left:10px; 会导致宽度也增加10px ）
  8. 选择器a b代表a中的b，而a,b才代表a和b都具备相应样式
  9. float会挨着前一个float对象元素进行float，而absolute在不设置偏移的情况下会抢占前一个absolute的元素的左上角的位置，把前一个元素给遮盖
  10. absolute属性在没有设置偏移量的时候，出现在最初的位置，这个最初的位置是指父级元素的内容左上角（会根据父元素的padding等属性可能离父元素有距离），这里可以视作偏移的默认位置，之后设置偏移量，按照祖先元素中有定位的作为参考点，哪个方向设置多少就距离那个参考点多少，没有设置的方向，则就是默认的值，fixed与其基本相同，只是参考点一定为html文档
  11. 行内元素没有width、height两个属性，margin和padding也只有左右方向有效果，上下都无效，设置其盒子高度可以通过line-height来设置

  12. 垂直方向margin属性的重叠
    已知 div 高 35px，紧邻其后的 X 元素上外边距为 15px，为该 div 设置 fixed 定位后，div与浏览器可视窗口上边界相距15px。
    为什么呢？
    因为X元素15px的上外边距与body的上外边距信息重叠了，所以body也跟着下移；然后，div在父元素body内，且没有设置偏移量，所以div相对其父元素body进行绝对定位（？）。

  13. 要使得ul水平，不仅可以利用float:left 或者absolute方法，也可以使用diplay:inline-block的方法来使得ul列表水平排列

  14. textarea固定大小/禁止拖动的方式
    resize: none; //彻底关掉拖动功能
    width:xxpx height:xxpx max-width:xxpx max-height:xxpx //不能拖动但是拖动图标还在

  15. 对于position为absolute或者fixed的标签元素，常常使用left: 50%; margin-left: xxpx;的技巧对他进行定位操作
    //html5实现锚点只能使用href=”#id”,不再使用name了
    //input标签是一个自闭合的标签，和button标签相比有很多缺点：比如不能嵌入其他元素

  16. 背景图片必须设置了宽度和高度才会显示出来，并且背景图片不像img标签那样可以伸缩。在不考虑浏览器兼容性的条件下，实现背景图片自适应窗口大小可以采用background-size属性，如:

    background-image: url(a.jpg);
    width: 1000px;
    height: 800px;
    background-size: 100% auto; //横向100% 纵向自适应

  17. 对于chrome来说，它的input标签在点击的时候都会有一个外部border和发光效果，这时候可以通过设置outline属性为空来取消这个效果

  18. 在css中，widht和height指的是内容区域的宽度和高度。增加内边距，边框和外边距不会影响内容区域的尺寸，但是会增加元素框的总尺寸。

  19. 浮动会让元素塌陷。即被浮动元素的父元素不具有高度。例如一个父元素包含了浮动元素，它将塌陷具有零高度。你可以按以下方法处理：

    在浮动元素后加个div设置clear: both; height:0,overflow:hidden
    使用clearfix;
    设置父元素浮动;

  20. ie6不支持min-height一类的属性，这时候要做css hacking，即使用_hright属性。使用min-height属性是为了在浮动或者决定定位的会级元素没有内容时也能有一定的高度而不至于塌下去

  21. 当既有背景图片，又有背景颜色的时候，并且背景图片不重复，就在左边显示背景图片，没有背景图片的地方显示背景颜色。这可以通过盒子模型的3d图来理解
